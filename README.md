## Description:
Step into the realm of the Programming Languages Quiz—an intricately curated and educational gaming experience engineered to scrutinize your foundational comprehension of three extensively-utilized programming languages: C++, Java, and Python.

## Features:
- Delve into a challenging array of Multiple Choice Questions (MCQs), meticulously tailored for each unique programming language.
- Navigate an intuitively designed interface, meticulously crafted using VB.NET within Visual Studio Community 2022.
- Seamlessly integrate with Microsoft SQL Server Management Studio version 19 for the adept management of data.

## Requirements:
- Operating System: Windows.
- .NET Framework.

## Installation:
Embark on the journey to set up the quiz game on your system:
1. Acquire the source code files.
2. Validate the configuration of your development environment, ensuring compatibility with Visual Studio Community 2022 and the requisite .NET Framework.
3. Unveil the project solution file (DevQuiz.sln) within Visual Studio.
4. Initiate the build and execution of the project.

## How to Play:
1. Initiate the quiz game application.
2. Select the programming language you desire to undergo assessment.
3. Tackle the MCQs with utmost precision.
4. Instantly receive feedback on your responses.
